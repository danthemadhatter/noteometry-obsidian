# Noteometry – Obsidian Plugin

Ink-first STEM notebook. Pencil + eraser + Math & Circuit Sniper. Powered by Excalidraw + Gemini 3.1 Pro Preview.

## Features
- **Three modes**: Text • Math • Circuits (Circuit Sniper)
- **READ INK** – Lasso handwriting → Gemini 3.1 Pro Preview → clean LaTeX
- **SOLVE** – DLP v12 step-by-step solutions
- **Math symbol palette** + **Circuit Sniper palette** (AC/DC symbols)
- MCM clean minimal design (no pressure sensitivity, no brushes)
- Stores as native `.excalidraw` files in your vault
- Secure Gemini key via Obsidian Secrets (`noteometrygeminiapi`)

Made for engineers who want a pencil, an eraser, and Noteometry.

## Installation
1. Download or clone this repo into your vault's `.obsidian/plugins/noteometry` folder
2. Enable "Noteometry" in Obsidian → Community plugins
3. Add your Gemini API key in Obsidian → Settings → Secrets (ID: `noteometrygeminiapi`)
4. Click the pencil ribbon icon
